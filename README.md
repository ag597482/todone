# Todone

Spring Boot application with Hello World API, Swagger (OpenAPI), and CORS support.

## Quick links

| Resource | URL |
|----------|-----|
| **Local API (Hello)** | [http://localhost:8080/api/hello](http://localhost:8080/api/hello) |
| **Swagger UI** | [http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html) |
| **OpenAPI JSON** | [http://localhost:8080/v3/api-docs](http://localhost:8080/v3/api-docs) |

## Run the application

```bash
./mvnw spring-boot:run
```

Or run the main class `com.indra.todone.TodoneApplication` from your IDE.

Ensure the app is running before opening the links above.
